🚀 QUICK START — v5

1. Collection Identity

Set project name + total batch size.

2. Upload Images

Each image gets a quantity slider. Auto-Distribute splits evenly. Total must equal batch size.

3. Audio Library

Add your track filenames once — e.g. 1.mp3, rare.mp3. Then on each image card, pick which tracks belong to that image from a dropdown. No file upload needed — XRP Cafe handles IPFS.

4. Traits

Define traits once (e.g. Hat → Gold, Silver). Then on each image card, pick which value applies. The rarity comes from the image's quantity — no extra math needed.

5. Preview & Export

Use the Batch Navigator to flip through every NFT before exporting. Download ZIP → drop files in media/ → upload to XRP Cafe.
